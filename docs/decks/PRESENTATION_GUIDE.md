# Presentation Guide

## Files

- `presentation-v2.html` - The main presentation file (TED Talk style, open in any web browser)
- `presentation-v2-outline.md` - Detailed slide-by-slide outline for reference

## Design Features

1. **Dark Background** - Professional black theme with high contrast
2. **Large Typography** - Headlines up to 4.5em, body text 1.4em for visibility
3. **Color Accents** - Orange (#FF6B35), Teal (#4ECDC4), Yellow (#FFE66D) for emphasis
4. **Visual Cards** - Information organized in card layouts with colored borders
5. **Stat Boxes** - Large numbers for key metrics (Precision, Recall, F1, Latency)
6. **Minimal Text** - Bullet points replaced with visual elements and cards
7. **Professional Spacing** - Generous whitespace for clarity

### Included Diagrams

- **System Architecture** (Slide 7) - Dual-pipeline architecture diagram
- **Scalability Analysis** (Slide 15b) - Performance charts showing 20 vs 1000 device scaling
